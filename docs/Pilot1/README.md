# Pilot1

Pilot1 is the first reference implementation of the Curriculum Architecture
Kit.

The pilot target is **AWS Certified Generative AI Developer - Professional
(AIP-C01)**. All Pilot1 implementation artifacts live under:

`docs/Pilot1/aip-c01/`

Reusable methodology, templates, and lessons learned belong in:

`docs/curriculum-architecture-kit/`

Do not put subject-specific Pilot1 materials into the generic kit unless the
material has been abstracted into reusable guidance.
