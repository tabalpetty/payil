# Reviewed Day 1 Question Items

Store normalized and reviewed Day 1 question items here.

An item can move toward approval only after:

- it follows `question-bank-specification.md`;
- the official exam code is `AIP-C01`;
- the question maps to a `Day01-orderNNN` topic;
- the correct answer is source-checked;
- distractors are plausible but defensibly wrong;
- remediation points to a Day 1 study guide, slide, or artifact.
