# Pilot1 AIP-C01

This directory is the clean artifact home for the Pilot1 implementation:

**AWS Certified Generative AI Developer - Professional (AIP-C01)**.

## Artifact Areas

- `curriculum-model/`: application of the generic architecture method to the
  AIP-C01 exam domain.
- `teacher-kit/`: instructor materials, teaching runbooks, facilitation guides,
  rubrics, and remediation guidance.
- `student-kit/`: learner-facing explanations, activities, labs, retrieval
  materials, assessments, and remediation.
- `exam-prep/`: exam-format summary, question-bank schema, tagged questions,
  timed practice, simulated exams, and readiness checks.
- `source-material/`: Pilot1 AIP-C01 source syllabi, prior planning artifacts,
  trusted reference inputs, and generated planning views.

## Rebuild Rule

Regenerate materials here from current official AWS AIP-C01 sources and trusted
project source material. Keep source traceability from exam guide to curriculum
unit, teaching activity, learner artifact, assessment, remediation, and
exam-prep item.
