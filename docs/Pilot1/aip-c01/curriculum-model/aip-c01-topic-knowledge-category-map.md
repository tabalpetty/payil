# AIP-C01 Topic Knowledge Category Map

This table lists the ordered AIP-C01 curriculum topics, places each topic into the accelerated 7-day curriculum, classifies each topic using the corrected two-axis language, and proposes the learner artifact to create for each topic based on the identified category/type mix.

Source: `docs/Pilot1/aip-c01/source-material/AIP-C01_Topics_Learning_Order_With_Knowledge_Type.xlsx`.

## Day And Ordering Rule

The `Curriculum Order` column uses `DayNN-orderNNN`. The day assignment follows the accelerated 7-day curriculum, while the within-day order preserves the prerequisite-oriented topic order from the source workbook: foundations first, then capabilities built on top of those foundations.

| Day | Topic range | Curriculum focus |
|---:|---|---|
| 1 | 001-010 | GenAI and AWS foundations, invocation, prompt baseline, basic evaluation |
| 2 | 011-020 | Data preparation, embeddings, vector stores, basic semantic retrieval, basic RAG |
| 3 | 021-038 | Advanced retrieval, RAG troubleshooting, prompt systems, FM API patterns, resilience |
| 4 | 039-073 | Model routing, deployment, application/enterprise integration, tools and agents |
| 5 | 074-095 | Safety, security, privacy, governance, and responsible AI |
| 6 | 096-128 | Cost, performance, operations, monitoring, evaluation, and troubleshooting |
| 7 | 129-133 | Final integration, ADRs, production readiness, mixed-domain exam scenarios |

## Category Mapping

| Knowledge Category | Knowledge Types | Typical Artifact Family |
|---|---|---|
| Knowledge | Declarative, Conceptual, Conditional, Causal, Normative | Recall sheet, concept map, decision table, cause-effect worksheet, policy/risk review |
| Skill | Procedural, Diagnostic, Strategic, Metacognitive | Lab/runbook, troubleshooting worksheet, architecture decision record, error log |
| Representation / Location | Explicit, Tacit, Implicit, Embedded, Institutional, Collective | Reference checklist, critique notes, habit rubric, configuration record, RACI/control map |
| Social / Organizational | Social, Relational | Stakeholder map, ownership register, escalation path |
| Embodied Skill | Embodied | Practice log, performance checklist, coaching notes |

## Artifact Selection Rule

The artifact column is derived from the topic types. When a topic has multiple types, the artifact combines the highest-impact evidence forms, prioritizing diagnostic, strategic, procedural, embedded, institutional, normative, conditional, causal, conceptual, and recall evidence in that order.
