# Source-To-Decomposition Deferrals

This file records official AIP-C01 objectives that are intentionally deferred,
omitted, or handled outside the main curriculum decomposition.

Layer 0 treats an official objective as acceptable only when it is either:

- explicitly covered by the decomposition; or
- explicitly deferred here with a reason and a planned handling path.

Do not use this file to hide accidental gaps. A deferral should explain why the
objective is not represented in the current topic map and where the learner
will encounter it later, if at all.
