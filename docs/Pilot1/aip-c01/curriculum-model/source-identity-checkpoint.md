## Guardrails

- Do not use AIF-C01 or AWS Certified AI Practitioner materials for Pilot1
  except as an explicit comparison.
- Treat workbooks as planning views, not the sole source of truth.
- Verify technical claims against current official AWS sources before
  publishing teacher, student, lab, or exam-prep materials.
- Preserve traceability from official exam task or skill to unit, activity,
  artifact, assessment, remediation, and exam-prep item.
