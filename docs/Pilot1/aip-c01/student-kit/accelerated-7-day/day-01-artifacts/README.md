
## How To Use

1. Read the Day 1 study guide.
2. Complete these focused artifacts in order.
3. Summarize the evidence in the parent Day 1 roll-up workbook.
4. Record misses or weak answers in the daily error log.

## Audit Rule

For a focused artifact to count as complete, it must include:

- the learner's own explanation or decision;
- at least one scenario-specific example;
- evidence matching the knowledge type;
- a self-check or gap note.
