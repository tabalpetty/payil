# AIP-C01 Accelerated 7-Day Student Artifacts

These artifacts turn the accelerated 7-day plan into learner work. Complete
them in order. The goal is not to produce polished documents; the goal is to
create evidence that you can explain, choose, build, diagnose, and remediate.

Start with [OPEN_FIRST.md](OPEN_FIRST.md).

If you are running the first pilot by yourself, also use
[PILOT_RUNBOOK.md](PILOT_RUNBOOK.md).

For actual reading and presentation material, use:

- [study-guides/README.md](study-guides/README.md)
- [presentations/README.md](presentations/README.md)

## How To Use This Kit

1. Complete the entry diagnostic before Day 1.
2. Complete one daily artifact per day.
3. Maintain the daily error log after every timed practice set.
4. Carry unresolved severe gaps into the next day.
5. Use Day 7 to decide honestly: `Ready`, `Remediate`, or `Do not schedule`.

## Completion Standard

For a day to count as complete:

- the daily artifact is filled with scenario-specific decisions;
- at least one decision includes rejected alternatives;
- safety, security, governance, or operations implications are addressed;
- timed practice has been completed;
- misses are classified in the error log;
- severe gaps have a named remediation action.
