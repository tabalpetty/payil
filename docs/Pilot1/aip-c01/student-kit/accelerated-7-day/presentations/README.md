# Accelerated 7-Day Presentations

These are slide-style outlines. They are designed for quick review before the
daily artifact work.

Use them like this:

1. Read the day's study guide.
2. Review the matching presentation.
3. Reconstruct the slide flow from memory.
4. Fill the daily artifact.
