## Remediation Menu

| Gap type | Repair action |
|---|---|
| Recall | Make retrieval cards, close notes, retry from memory. |
| Conceptual | Rebuild the diagram and explain relationships aloud. |
| Procedural | Complete a worked example, then an independent variant. |
| Conditional | Compare two similar cases where different choices are correct. |
| Causal | Use predict-observe-explain or cause/evidence table. |
| Diagnostic | Identify evidence before proposing fixes. |
| Governance | Add owner, control, evidence, approval, and residual risk. |
| Metacognitive | Compare confidence against actual result and revise study strategy. |
