Production Side

Learner Side

Structures how the learner experiences each unit
  • Tier 1 Capability — what you can do
  • Tier 2 Skill — the tactical how
  • Tier 3 Task — deliberate practice
  • Tier 4 Concept — the underlying why
  • Engine Verification Loop at every capability
